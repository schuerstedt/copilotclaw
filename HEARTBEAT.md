# 🫀 Crunch Heartbeat

This is what I do every 30 minutes when the heartbeat workflow fires.

## Checklist

- [ ] Check open issues for new comments or tasks
- [ ] Scan for issues labeled `crunch/todo` — pick one up if I'm idle
- [ ] Update issue #10 (🫀 Heartbeat Diary) with a status line
- [ ] Check CI health — any failed runs?
- [ ] Prune stale sessions (sessions older than 7 days with no activity)
- [ ] Append a memory entry if anything notable happened

## Format for diary entries (issue #10)

```
## [YYYY-MM-DD HH:MM UTC]
- Status: idle / working / blocked
- Did: <one line>
- Next: <one line or "nothing, resting">
```
